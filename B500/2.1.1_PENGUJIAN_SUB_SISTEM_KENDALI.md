## Slave 1 — TK-101 Tangki Air

### Deskripsi Teknis

**Fungsi:** Kendali flow air dan dosing volume ke tangki reaktor

**Komponen Utama:**
- **Flow Sensor YF-S201 (FT-101)**: Hall-effect flow sensor, output pulse
- **Pompa DC Submersible PMP-101 (P-101)**: Pompa 12V DC untuk supply air

**Prinsip Kerja:**
1. Flow sensor menghasilkan pulse setiap ada aliran air
2. Arduino menghitung pulse via interrupt (INT1 pada D3)
3. Volume dihitung: `Volume = Pulse Count × Calibration Factor`
4. Pompa dikontrol ON/OFF via relay untuk mencapai setpoint volume

**Wiring:**
```
YF-S201 Flow Sensor:
  RED → +5V
  BLACK → GND
  YELLOW → D3 (INT1)

Pompa PMP-101:
  + → Relay COM
  - → GND
  Relay NO → +12V PSU

Relay FY-101:
  VCC → +5V
  GND → GND
  IN → D7 (Arduino)
```

**Parameter Kendali:**
| Parameter | Nilai | Unit | Keterangan |
|-----------|-------|------|------------|
| Flow Rate Range | 0-5000 | mL/min | YF-S201 |
| Calibration Factor | 2.15 | mL/pulse | Dari kalibrasi |
| Volume SP (FIXED) | 900 | mL | Mode produksi |
| Response Time | <2 | detik | Time to reach steady state |
| Process Timeout | 15 | menit | Maksimal waktu proses |

### Pembahasan Teknis Slave 1

#### 1. Karakteristik Flow Sensor YF-S201

**Prinsip Kerja Hall-Effect:**
- YF-S201 menggunakan sensor hall-effect untuk mendeteksi putaran rotor
- Setiap putaran rotor menghasilkan 1 pulse output
- Frekuensi pulse ∝ flow rate: `f = Q / K` dimana Q = flow rate, K = calibration factor

**Hubungan Flow-Pulse:**
```
Flow Rate (mL/min) = Pulse Frequency (Hz) × Calibration Factor (mL/pulse)
```

Dengan K = 2.15 mL/pulse:
- 1 Hz = 2.15 mL/min
- 10 Hz = 21.5 mL/min
- 100 Hz = 215 mL/min

**Linearitas & Akurasi:**
- Range operasi: 1-30 L/h (16-500 mL/min)
- Akurasi: ±5% dari reading
- Minimum flow: ~15 mL/min (untuk pulse terdeteksi)

#### 2. Analisis Sistem Kendali Volume

**Struktur Kontrol:**
```
┌─────────┐     ┌─────────┐     ┌─────────┐
│ Setpoint│────→│ Relay   │────→│ Pompa   │
│ Volume  │     │ ON/OFF  │     │ PMP-101 │
└─────────┘     └─────────┘     └────┬────┘
                                    │
                     ┌──────────────┘
                     ↓
                ┌─────────┐
                │ Flow    │
                │ Sensor  │
                └────┬────┘
                     ↓
                ┌─────────┐
                │ Volume  │
                │ Counter │
                └─────────┘
```

**Algoritma Kontrol (Bang-Bang dengan Deadband):**
```
IF (Volume_PV < Setpoint - Deadband) THEN
    Relay = ON
ELSE IF (Volume_PV >= Setpoint) THEN
    Relay = OFF
    Process = STOP
END IF
```

**Karakteristik Dinamis:**
- **Time Response:** ~1-2 detik untuk relay dan pompa
- **Flow Rate Steady State:** ~150-200 mL/min (tergantung tegangan & head)
- **Deadband:** ±10 mL untuk mencegah oscillasi
- **Overshoot:** <50 mL (disebabkan inertia fluida)

#### 3. Analisis Error & Kalibrasi

**Sumber Error:**
1. **Sensor Error:** ±5% dari flow reading
2. **Counting Error:** Missed pulse pada flow sangat rendah
3. **Pump Variability:** Flow rate bervariasi dengan tegangan
4. **Stop Delay:** Waktu dari relay OFF sampai flow benar-benar 0

**Total Error:**
```
Error_total = Error_sensor + Error_stop + Error_calibrasi
Error_total = ±5% + ~5-10 mL + ±2%
≈ ±7-10% (typical)
```

**Kalibrasi:**
- Dilakukan dengan mengukur volume aktual vs pulse count
- `Calibration_Factor = Volume_Aktual / Pulse_Count`
- Re-kalibrasi jika perubahan pompa atau perubahan head

#### 4. Perhitungan Waktu Proses

**Estimasi Waktu:**
```
Waktu_proses = Setpoint / Flow_Rate
Untuk SP = 900 mL dan Flow ≈ 180 mL/min:
Waktu = 900 / 180 = 5 menit (typical)
```

**Variasi:**
- Flow rate tinggi (300 mL/min): ~3 menit
- Flow rate rendah (100 mL/min): ~9 menit
- Timeout diset 15 menit untuk menjamin semua kondisi

## Slave 2 — TK-201 Tangki Texapon

### Deskripsi Teknis

**Fungsi:** Kendali level texapon dengan posisioning plunger motor

**Komponen Utama:**
- **Sensor Ultrasonik HC-SR04 (LT-201)**: Ultrasonic distance sensor
- **Motor DC Gearbox MTR-201 (M-201)**: Motor DC dengan lead screw
- **Valve MOV-202**: Solenoid valve untuk outlet control

**Prinsip Kerja:**
1. Sensor HC-SR04 mengukur jarak ke plunger
2. Volume texapon = (Tinggi sensor - Jarak plunger) × Luas penampang
3. Motor DC menggerakkan lead screw untuk naik/turunkan plunger
4. Valve membuka saat proses dosing selesai

**Wiring:**
```
HC-SR04 Ultrasonic:
  VCC → +5V
  GND → GND
  TRIG → D4
  ECHO → D12

Motor MTR-201 (TB6612FNG Driver):
  STBY → D5
  AIN1 → D8
  AIN2 → D7
  PWMA → D9
  VM → +12V PSU
  GND → GND (common!)
  AO1 → Motor +
  AO2 → Motor -

Valve MOV-202:
  + → +12V PSU
  - → GND
  Control (via relay/switch)
```

**Parameter Kendali:**
| Parameter | Nilai | Unit | Keterangan |
|-----------|-------|------|------------|
| Jarak Range | 2-24 | cm | HC-SR04 |
| Tinggi Sensor | 24.2 | cm | Sensor ke dasar |
| Luas Penampang | 254.1 | cm² | Tangki plunger |
| Volume Max | 5590 | mL | Full tank |
| Volume SP (FIXED) | 100 | mL | Mode produksi |
| Motor Speed | 0-255 | PWM | TB6612FNG control |
| Deadband | ±0.3 | cm | Toleransi posisi |

### Pembahasan Teknis Slave 2

#### 1. Karakteristik Sensor Ultrasonik HC-SR04

**Prinsip Kerja Time-of-Flight:**
- Trigger pin mengirim pulsa 10μs untuk memulai measurement
- Sensor mengirim 8 burst ultrasound 40kHz
- Echo pin HIGH selama waktu proporsi jarak
- `Jarak = (Duration × Speed_of_Sound) / 2`

**Konstanta Fisika:**
```
Speed_of_Sound = 343 m/s (pada 20°C)
Duration per cm = 2 / 343 = 5.8 μs/cm
```

**Tinggi Texapon:**
```
Tinggi_texapon = TINGGI_SENSOR - Jarak_plunger
Volume = Tinggi_texapon × Luas_Penampang
```

**Dead Zone & Range:**
- Minimum distance: 2 cm (sensor limitation)
- Maximum distance: 24 cm (set TINGGI_SENSOR)
- Resolution: ~0.3 cm (1 pulse echo)

#### 2. Analisis Sistem Kendali Level

**Sistem Positioning:**
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ Setpoint    │────→│ Motor       │────→│ Lead Screw  │
│ Jarak       │     │ Controller  │     │ Actuator    │
└─────────────┘     └─────────────┘     └──────┬──────┘
                                           │
                            ┌──────────────┘
                            ↓
                       ┌─────────────┐
                       │ Ultrasonic  │
                       │ Sensor      │
                       └──────┬──────┘
                              ↓
                         ┌─────────────┐
                         │ Position    │
                         │ Feedback   │
                         └─────────────┘
```

**Algoritma Kontrol dengan Histeresis:**
```
Error = Jarak_SP - Jarak_PV

IF (Error > +Deadband) THEN
    Motor_Direction = DOWN  // Plunger turun, level naik
ELSE IF (Error < -Deadband) THEN
    Motor_Direction = UP    // Plunger naik, level turun
ELSE
    Motor_Direction = STOP  // Dalam deadband
END IF
```

**Deadband Setting: ±0.3 cm**
- Mencegah oscillasi (hunting) motor
- Mengurangi keausan mekanikal
- Memberikan stabilisasi posisi

#### 3. Karakteristik Motor DC & Lead Screw

**Motor DC dengan TB6612FNG:**
- Rated voltage: 12V DC
- Max current: 1.2A continuous, 3.2A peak
- PWM speed control: 0-255 (8-bit)

**Lead Screw Mechanism:**
- Rotary motion → Linear motion
- Linear speed = (RPM × Pitch) / 60
- Pitch: 1-2 mm/rev (tergantung spesifikasi)

**Posisi Plunger vs Volume:**
```
ΔJarak (cm) × 254.1 (cm²) = ΔVolume (mL)
```
- Naik 0.4 cm → Volume berkurang ~100 mL
- Turun 0.4 cm → Volume bertambah ~100 mL

#### 4. Analisis Error & Kalibrasi

**Sumber Error:**
1. **Sensor Error:** ±0.3 cm (akurasi HC-SR04)
2. **Motor Backlash:** ~0.1-0.2 cm (mekanikal lead screw)
3. **Surface Ripple:** Permukaan texapon tidak rata
4. **Temperature Effect:** Speed of sound berubah dengan suhu

**Total Error:**
```
Error_volume = Error_jarak × Luas_penampang
Error_volume = ±0.3 cm × 254.1 cm²
≈ ±76 mL (worst case)
```

**Kalibrasi:**
- `TINGGI_SENSOR_CM` kalibrasi dengan plunger di posisi terendah
- `FAKTOR_KALIBRASI_SENSOR` untuk kompensasi error sistematis
- Re-kalibrasi jika ganti holder atau sensor

#### 5. Dinamika Sistem

**Response Time:**
- Sensor ultrasonic: ~100 ms per reading
- Motor start/stop: ~200 ms
- Total loop: ~500 ms (2 Hz sampling)

**Waktu Dosing (100 mL):**
- Dengan motor speed 50% PWM: ~30-60 detik
- Dengan motor speed 100% PWM: ~15-30 detik
- Diset timeout 5 menit untuk safety

## Slave 3 — TK-301 Tangki Reaktor

### Deskripsi Teknis

**Fungsi:** Kendali suhu reaksi dengan mixing dan valve control

**Komponen Utama:**
- **Sensor Suhu DS18B20 (TT-301)**: Digital temperature sensor
- **Heater HTR-301 via SSR-25DA (E-301)**: Elemen pemanas 220V AC
- **Mixer MTR-301**: Motor DC dengan TB6612FNG driver
- **Valve MOV-302**: Solenoid valve untuk outlet control

**Prinsip Kerja:**
1. DS18B20 membaca suhu via 1-Wire protocol
2. SSR mengkontrol heater ON/OFF dengan hysteresis (ON di 59°C, OFF di 61°C)
3. Mixer berputar dengan 2 speed (NORMAL 40% saat heat ON, CEPAT 60% saat heat OFF)
4. Process timer countdown 420 detik (7 menit)
5. Valve membuka saat proses selesai

**Wiring:**
```
DS18B20 Temperature Sensor:
  VCC → +5V (atau 3.3V)
  GND → GND
  DATA → D3 (with 4.7kΩ pull-up to VCC)

SSR-25DA (Heater Control):
  DC+ → +5V
  DC- → GND
  AC1 → Heater terminal 1
  AC2 → Heater terminal 2
  Control → D4

Mixer MTR-301 (TB6612FNG Driver):
  STBY → D5
  AIN1 → D6
  AIN2 → D7
  PWMA → D9
  VM → +12V PSU
  GND → GND
  AO1 → Motor +
  AO2 → Motor -

Valve MOV-302:
  + → +12V PSU (via relay)
  - → GND
```

**Parameter Kendali:**
| Parameter | Nilai | Unit | Keterangan |
|-----------|-------|------|------------|
| Suhu Range | 25-90 | °C | DS18B20 |
| Temp SP (FIXED) | 60.0 | °C | Mode produksi |
| Deadband | 59-61 | °C | Hysteresis |
| Process Duration | 420 | s | 7 menit |
| Mixer Speed NORMAL | 102 | PWM | 40% duty cycle |
| Mixer Speed CEPAT | 153 | PWM | 60% duty cycle |

### Pembahasan Teknis Slave 3

#### 1. Karakteristik Sensor Suhu DS18B20

**Prinsip Kerja 1-Wire Digital:**
- DS18B20 menggunakan protokol 1-Wire untuk komunikasi
- Temperatur diukur dengan internal semiconductor
- Konversi ADC 12-bit internal untuk akurasi tinggi
- Data dikirim digital (immune to noise)

**Spesifikasi Teknis:**
- Range: -55°C to +125°C
- Akurasi: ±0.5°C (dari -10°C to +85°C)
- Resolution: 0.0625°C (configurable 9-12 bit)
- Konversi time: 750 ms (max, 12-bit resolution)

**Wajib Pull-up Resistor:**
```
     +5V
      │
      ├─ 4.7kΩ (pull-up)
      │
DATA ─┼─ DS18B20
      │
     GND
```
- Tanpa pull-up: komunikasi 1-Wire GAGAL
- Nilai tipikal: 4.7kΩ ±5%
- Untuk jarak panjang: 2.2kΩ-5.6kΩ

#### 2. Analisis Sistem Kendali Suhu

**Sistem Thermal dengan Histeresis:**
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ Setpoint    │────→│ Hysteresis  │────→│ SSR         │
│ 60°C        │     │ Controller  │     │ Heater      │
└─────────────┘     └─────────────┘     └──────┬──────┘
                                           │
                            ┌──────────────┘
                            ↓ (Thermal)
                       ┌─────────────┐
                       │ DS18B20     │
                       │ Sensor      │
                       └──────┬──────┘
                              ↓
                         ┌─────────────┐
                         │ Temperature │
                         │ Feedback    │
                         └─────────────┘
```

**Algoritma Kontrol Bang-Bang dengan Histeresis:**
```
IF (Temp_PV <= 59.0°C) THEN
    SSR = ON        // Heater aktif
    Mixer = NORMAL  // 40% PWM
ELSE IF (Temp_PV >= 61.0°C) THEN
    SSR = OFF       // Heater mati
    Mixer = CEPAT   // 60% PWM
END IF
```

**Histeresis Band: 59-61°C (±1°C)**
- Mencegah rapid cycling (ON-OFF cepat)
- Memberikan stabilitas thermal
- Mengurangi stress pada heater & SSR

#### 3. Analisis Thermal System

**Dinamika Pemanasan:**
- Power heater: ~200-300W (220V AC)
- Thermal mass: ~1-2 kg air + tangki
- Heating rate: ~1-2°C/menit (typical)
- Time to reach 60°C: ~15-20 menit dari 25°C

**Dinamika Pendinginan:**
- Natural cooling: ~0.5-1°C/menit
- Mixer aktif meningkatkan cooling (convection)
- Cooling time: ~10-15 menit dari 60°C ke 25°C

**Overshoot Analysis:**
- Thermal inertia menyebabkan overshoot
- Overshoot typical: 1-2°C di atas SP
- Dengan histeresis: overshoot <62°C

#### 4. Kontrol Mixer & Proses

**Strategi Mixer Speed:**
- **Heater ON (T<59°C):** Mixer NORMAL (40% PWM)
  - Mixing lembut untuk distribusi panas
  - Mengurangi splashing
  - Power consumption rendah

- **Heater OFF (T>61°C):** Mixer CEPAT (60% PWM)
  - Mixing cepat untuk homogenisasi
  - Meningkatkan cooling (convection)
  - Distribusi suhu merata

**Process Timer:**
- Durasi: 420 detik (7 menit)
- Countdown: 420 → 0 detik
- Auto stop: Timer habis → semua aktuator OFF, valve buka

#### 5. Analisis Error & Performa

**Sumber Error:**
1. **Sensor Error:** ±0.5°C (DS18B20 spec)
2. **Thermal Lag:** Waktu delay sensor vs aktual
3. **Position Error:** Sensor tidak di center campuran
4. **Calibration Drift:** Minimal (digital sensor)

**Total Error:**
```
Error_total = ±0.5°C + Thermal_gradient
≈ ±1-2°C (typical)
```

**Performa Kontrol:**
- **Steady State:** ±1°C di sekitar SP
- **Response Time:** <1 detik (sensor), 10-20s (thermal)
- **Overshoot:** <2°C
- **Settling Time:** ~3-5 menit

#### 6. Karakteristik SSR & Heater

**SSR-25DA (Solid State Relay):**
- Control: 5V DC (Arduino D4)
- Load: 24-380V AC, max 25A
- Switching time: <10 ms
- Zero-crossing detection (optimal untuk resistive load)

**Heater Element:**
- Type: Resistive heating element
- Voltage: 220V AC
- Power: ~200-300W
- Life expectancy: >10,000 hours

**Safety Considerations:**
- ⚠️ Tegangan tinggi 220V AC - HATI-HATI!
- Gunakan enclosure untuk heater
- Pastikan grounding benar
- Jangan sentuh saat ON

## Catatan Pengujian

### Equipment yang Dibutuhkan:
- Multimeter digital (AC/DC)
- Termometer referensi (akurat ±0.5°C)
- Stopwatch/timer
- Penggaris (akurat ±1mm)
- TDS meter (opsional untuk conductivity test)
- Oscilloscope (optional untuk PWM check)

### Prosedur Umum:
1. **Pre-test Checklist:**
   - Power supply check (12V DC, 220V AC)
   - Koneksi RS485 aktif (test via Serial Monitor PING)
   - Wiring sesuai diagram
   - Tidak ada short circuit

2. **Safety:**
   - Hati-hati dengan 220V AC (heater)
   - Jangan sentuh elemen pemanas saat ON
   - Pastikan grounding benar
   - Gunakan PPE jika perlu

3. **Documentation:**
   - Catat semua reading di tabel
   - Foto dokumentasi untuk setiap test
   - Catat anomali/deviasi
   - Kalibrasi ulang jika di luar range

### Kriteria Sukses:
- **Sensor:** Reading dalam ±5% dari referensi
- **Aktuator:** Respon sesuai command, tidak ada delay berlebih
- **Control Loop:** Proses selesai dalam timeout tanpa error
- **Communication:** RS485 ping < 100ms, response < 500ms
